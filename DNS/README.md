# Mise en place d'un serveur DNS

## Objectif

Mettre en place un serveur DNS afin de permettre la résolution des noms de domaine au sein du réseau.

## Technologies utilisées

- Cisco Packet Tracer
- DNS
- IPv4
- TCP/IP

(Capture de la topologie à ajouter)

## Étapes réalisées

1. Configuration du serveur DNS
2. Création des enregistrements DNS
3. Configuration des postes clients
4. Vérification de la résolution des noms

## Résultat

Les postes clients peuvent accéder aux équipements du réseau en utilisant leur nom plutôt que leur adresse IP.

## Compétences développées

- Configuration DNS
- Résolution de noms
- Administration réseau
- Diagnostic réseau

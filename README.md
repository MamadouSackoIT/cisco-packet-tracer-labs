# Laboratoires Cisco Packet Tracer

Ce dépôt regroupe les laboratoires réalisés dans le cadre de mon apprentissage des réseaux et systèmes.

## Projets réalisés

### DHCP
Configuration d'un serveur DHCP permettant l'attribution automatique des adresses IP aux postes clients.

### DNS
Mise en place d'un serveur DNS afin de permettre la résolution de noms sur le réseau.

### Communication Réseaux
Configuration de routeurs et switchs afin de permettre la communication entre plusieurs équipements réseau.

## Compétences développées

- Adressage IPv4
- DHCP
- DNS
- Routage
- Switching
- Diagnostic réseau
- Tests de connectivité

## Outils utilisés

- Cisco Packet Tracer
- TCP/IP
- Routeurs Cisco
- Switchs Cisco
